# 一、标准库搭建

## 要点
- 向业务方简述业务逻辑，并确认标准库采用哪些官方网站
- 根据系统业务目标，设计标准库字段，开发爬虫获取数据

## 任务拆解
1. 找业务获取标准库搭建用的网站列表
2. 确认各网站的设计维度信息充足
3. 输出爬取文档，提供列表页地址与详情页框选截图
4. 设计标准库字段表（raw阶段，存储各网站的类目字段，用json存储商品原始数据）
5. 发布爬虫任务
6. 爬虫开发、验收
7. 搭建标准库（raw）
8. 完成符合阿里云图像搜索的打标（通过阿里云自身识别或另外调用LLM来实现，需测试准确率）
9. 设计结构化prompt，并将json中的原始数据拆解为结构化设计维度数据
10. 测试、优化prompt
11. 完成结构化AI接口文档
12. 结构化AI接口开发、测试、验收
13. 完成结构化处理，输出标准库（standard）

**交付：** 标准库来源网址列表 + 数据表设计 + 爬虫 + 商品标准数据库
